# 한국 Skills 마켓 구축

> 상태: 진행중
> 마지막 업데이트: 2026-04-16

## 개요
Claude Code Skills 마켓플레이스의 한국 버전. 한국어 큐레이션/번역으로 시작해서 한국어 전용 스킬 플랫폼으로 확장. 나중엔 세계관을 갖춰가는 방향.

---

## 방향 결정 (2026-04-16)

- **형태**: 커뮤니티 플랫폼 (단순 큐레이션 X)
- **수익화**: 초기엔 없이 시작, 나중에 붙임
- **콘텐츠 전략**: 글로벌 스킬 번역/큐레이션 → 한국어 스킬 전용 섹션 → 한국어 스킬 마켓
- **이름**: 스킬인벤 (skillinven)
- **도메인**: skillinven.kr / skillinven.io (확인 필요)

## 기술 스택 (추천안, coli 최종 결정)

| 영역 | 추천 | 이유 |
