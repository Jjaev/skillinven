## 스키마

```typescript
interface Skill {
  id: string               // github_owner/repo/skill-name
  name: string             // SKILL.md에서 파싱
  description_en: string   // SKILL.md 원문
  description_ko: string | null  // 수동 번역 입력
  is_reviewed: boolean     // 욱님 검수 여부
  github_url: string
  stars: number
  author: string
  category: string
  compatible_with: string[] // ['claude', 'codex', 'gemini', 'chatgpt']
  install_command: string
  updated_at: string
}
```

## 한국어 번역 입력 방법

초반엔 Supabase 대시보드에서 `description_ko` 직접 입력.
`is_reviewed: false` → 욱님이 검수 후 `true`로 변경.
검수 전엔 영문 표시, 검수 후 한국어 표시.
