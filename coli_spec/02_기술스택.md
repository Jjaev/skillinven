# 기술 스택

## 스택

| 영역 | 선택 | 이유 |
|------|------|------|
| 프레임워크 | Next.js 15 (App Router) | SSG, Vercel 배포 최적 |
| 스타일 | Tailwind CSS | 빠른 프로토타이핑 |
| DB | Supabase 무료티어 | 스킬 데이터 저장 |
| 배포 | Vercel | 무료, 자동 배포 |
| 번역 | 수동 (추후 Claude API) | 초반엔 욱님이 직접 |
