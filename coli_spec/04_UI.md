# UI / UX

## 페이지 구조

```
/              → 홈 (검색바 + 스킬 카드 목록)
/skills/[id]   → 스킬 상세
```

## 홈 페이지

- 상단: 검색바
- 하단: 스킬 카드 그리드

## 스킬 카드

```
┌─────────────────────┐
│ 스킬 이름           │
│ 한국어 설명 (짧게)  │
│ [Claude] [Codex]    │
│ ★ 42  github →     │
└─────────────────────┘
```

## 스킬 상세 페이지

- 이름 + 설명 (한국어 우선, 없으면 영문)
- 설치 방법:
```bash
# Claude Code
claude /install-skill {github_url}

# Codex CLI
codex skill install {github_url}
```
- GitHub 원본 링크

## 디자인 방향

**MVP 기준점: Notion Template Gallery 구조**
- 카드 그리드 레이아웃
- 카테고리 사이드바 or 탭 필터
- 상세 페이지 — 설명 + 미리보기 + 설치 가이드

깔끔하고 실용적으로. 다크/라이트 일단 라이트만.

**다음 단계 (MVP 이후)**
