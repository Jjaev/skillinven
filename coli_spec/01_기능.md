# 기능 명세

## MVP (이것만 먼저)

### P0 — 없으면 의미 없음
1. **스킬 목록 페이지** — 카드형, 이름/설명/GitHub 링크
2. **스킬 상세 페이지** — 설명, 설치 방법, 호환 에이전트
3. **기본 검색** — 이름 기반

### P1 — 차별화 핵심
4. **한국어 설명 표시** — `description_ko` 있으면 한국어, 없으면 영문 fallback
