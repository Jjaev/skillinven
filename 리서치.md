## 역설계 — 기존 마켓 구조

```
GitHub 레포 (SKILL.md 포함)
        ↓ 자동 스크래핑/인덱싱
    마켓 웹사이트
        ↓
   CLI 도구로 설치
```

핵심: 스킬 직접 안 만들어도 됨. GitHub에서 자동 수집.

| 마켓 | 기술 스택 | 차별화 |
|------|----------|--------|
| SkillsMP | 미상 | 최대 규모, 직업별 필터, 품질 필터(2스타↑) |
| SkillHub | React + Rust(Tauri) | 데스크탑 앱 + CLI |
| claudeskills.info | 미상 | 큐레이션, 심플 UX |
