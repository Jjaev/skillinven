# 스킬인벤 수익모델

> 작성: 뉴턴 | 2026-04-17

---

## 현실적인 로드맵

```
Phase 1 (MVP)     → 무료. 트래픽/스킬 수 쌓기
Phase 2 (성장)    → Carbon Ads 붙이기
Phase 3 (안정화)  → 프리미엄 리스팅 + 뉴스레터 스폰서
Phase 4 (확장)    → B2B / 파트너십
```

---

## 단계별 BM

### Phase 2 — 광고 (첫 수익)

| 방법 | 특징 |
|------|------|
| **Carbon Ads** | 개발자 사이트 전용 광고 네트워크. 영업 없이 신청만. |
| **EthicalAds** | 동일. 개발자 타깃 CPM 나쁘지 않음 |
| **스폰서드 카테고리** | "이 카테고리는 XX가 후원합니다" |

광고주 후보: Anthropic, AI 툴 회사들 — 개발자 타깃으로 딱 맞음.

### Phase 3 — 프리미엄 리스팅 + 뉴스레터

- **프리미엄 리스팅** — 무료 등록 + 상단 노출은 유료. 마찰 적고 심플.
- **주간 뉴스레터** — 한국 개발자 AI 툴 큐레이션 → 스폰서십

### Phase 4 — B2B / 파트너십

- 스킬 데이터 API 접근권 (B2B)
- 기업 온보딩 패키지 — "팀용 스킬 세팅"
- Anthropic 공식 한국 파트너 레버리지
