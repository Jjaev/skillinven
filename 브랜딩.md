## 세계관 방향 (결정 중)

### 방향 A — 무협 비급서
```
Skills = 비급 (숨겨진 최강 기술서)
사용자 = 무인 (강해지려는 자)
마켓 = 비급이 거래되는 장소
```
이름 후보: 비급각, 비기서, 비전서

### 방향 B — 갓슈벨 스펠북
```
Skills = 스펠북 (파트너가 읽어야 발동)
AI Agent = 마모노
사용자 = 파트너 (인간)
성장: 새 주문 습득할수록 강해짐
```
이름 후보: 스펠북, 주문서
- AI + 사람 협업 구조와 1:1 매핑 → 뉴턴 픽
- 성장/수집 세계관 → 커뮤니티 락인 강함

### 방향 C — 혼합 (동양 세계관 + 스킬서)
무협 느낌 + 스펠북 감성
